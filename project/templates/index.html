Smash bot finisher

Un challenge sur le jeu Super Smash Bros Ultimate où l'objectif est d'éradiquer tous les champions (IA) et qui comporte les règles suivantes :

  • - Choisir un seul personnage pour tout le challenge
  • - Choisir le stage "Champ de Bataille" en mode DF pour tout le challenge
  • - Le temps doit être de 6min
  • - Le nombre de vie doit être 3
  • - Le niveau de l'IA doit être 9
  • - Le personnage de l'IA doit être pris dans l'ordre donné par le site
  • - Impossible de prendre de l'avance sur l'ordre donné
  • - Les défaites doivent être annoncé sur le site
  • - Les victoires doivent être annoncé sur le site, accompagné d'un screenshot de l'écran de victoire
